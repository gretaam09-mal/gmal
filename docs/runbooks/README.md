# Runbooks

Operational how-tos. `local-development.md` covers running everything
locally; `clerk-setup.md`, `companies-house-setup.md`, and
`anthropic-setup.md` cover the external accounts/keys each integration
needs (none of them required to run the test suite); more runbooks
(deploys, incident response) land as the corresponding infrastructure
exists — see `infra/aws/main.tf`.
